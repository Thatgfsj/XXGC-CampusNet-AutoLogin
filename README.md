## 🤔 这是什么？
这个脚本可以自动完成校园网登录流程，第一次配置后，后续只需一键运行即可登录，无需重复输入账号密码。

## 🚀 使用教程
### 🔹 第一次使用（需完成初始配置）
#### 步骤1：运行脚本
两种方式任选其一：
- 找到 `xywdl.ps1` 文件 → 右键 → 选择「使用PowerShell运行」
- 直接双击运行 `联网程序.bat`（更简单，推荐小白使用）

#### 步骤2：获取登录信息
脚本会自动尝试获取登录信息，若提示“自动获取失败”，按以下步骤操作：
1. 断开当前校园网Wi-Fi，重新连接
2. 打开任意浏览器（Chrome/Edge/360等）
3. 在地址栏输入任意网址（如 www.qq.com）并回车
4. 浏览器会自动跳转到校园网登录页面
5. 复制地址栏里的**完整链接**（格式示例：`http://172.18.xxx.xxx:6060/portal.do?wlanuserip=...`）
6. 回到脚本窗口，粘贴链接并按回车

#### 步骤3：输入账号信息
按脚本提示逐步操作：
1. 选择运营商：输入 `1`（移动）/`2`（联通）/`3`（电信）→ 回车
2. 输入纯数字学号（示例：2025020301）→ 回车
3. 输入校园网密码（输入时看不到字符，正常现象）→ 回车
4. 再次输入密码确认 → 回车

#### 步骤4：保存配置
看到提示 `保存后下次无需重新输入 (y/N)` 时：
- 输入 `y` → 回车（推荐！下次可一键登录）
- 输入 `N` → 回车（下次需重新配置）

### 🔹 后续使用（一键登录）
配置完成后，每次登录只需：
1. 右键点击 `xywdl.ps1` → 「使用PowerShell运行」
   （或直接双击 `联网程序.bat`）
2. 等待3-5秒，脚本会自动完成登录
3. 看到「认证成功」提示即表示登录完成

## ❌ 常见问题解答（Q&A）
| 问题 | 解决方案 |
|------|----------|
| 脚本打不开？ | 右键文件 → 属性 → 勾选「解除锁定」→ 确定 → 重新运行 |
| 提示“无法运行脚本”？ | 1. 以管理员身份打开PowerShell<br>2. 输入命令：`Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser`<br>3. 按回车确认 |
| 忘记保存配置，下次要重新输入？ | 重新按「第一次使用」步骤操作，最后输入 `y` 保存即可 |
| 账号密码修改后登录失败？ | 1. 打开文件资源管理器，地址栏输入 `%APPDATA%`<br>2. 删除 `xxgc_campus_net_config.txt` 文件<br>3. 重新按「第一次使用」步骤配置 |
| 想删除已保存的配置？ | 1. 打开文件资源管理器，地址栏输入 `%APPDATA%`<br>2. 找到 `xxgc_campus_net_config.txt` 文件并删除 |
