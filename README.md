# 新乡工程学院校园网自动登录脚本
一键搞定校园网登录验证请求

## 📌 重要说明
本脚本仅适用于**新乡工程学院**校园网，其他学校请自行修改。
B站：https://www.bilibili.com/opus/1174460821448687621

这是什么？
这个脚本可以让你一键登录校园网，不用每次都手动输入账号密码。

第一次使用（需要设置）
步骤1：运行脚本
找到 xywdl.ps1 文件，右键点击，选择 "使用PowerShell运行"  
或者直接运行  联网程序.bat

步骤2：获取登录信息
脚本会尝试自动获取登录信息，如果失败，按下面操作：

1.
断开Wi-Fi，然后重新连接校园网
2.
打开浏览器（Chrome、Edge等）
3.
在地址栏输入任意网站
4.
浏览器会跳转到一个登录页面
5.
复制浏览器地址栏里的全部内容（类似：http://172.18.xxx.xxx:6060/portal.do?wlanuserip=...）
6.
回到脚本窗口，粘贴进去，按回车
步骤3：输入账号信息
按提示操作：

1.
选择运营商：输入 1（移动）、2（联通）或 3（电信），按回车
2.
输入学号（例如：2025020301），按回车
3.
输入密码，按回车
4.
再次输入密码确认，按回车
步骤4：保存配置
看到提示 保存后下次无需重新输入 (y/N) 时，输入 y 按回车。

以后使用（一键登录）
以后每次使用，只需要：

1.
右键点击 xywdl.ps1
2.
选择 "使用PowerShell运行"（或者直接运行联网程序.bat）
3.
等待几秒钟，脚本会自动登录


常见问题
Q：脚本打不开？
A： 右键点击文件 → 属性 → 解除锁定 → 确定，然后再运行。

Q：提示"无法运行脚本"？
A： 以管理员身份打开PowerShell，输入：

Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser
然后按回车。

Q：忘记保存了，下次还要重新输入？
A： 没关系，再按第一次的步骤重新来一遍，这次记得输入 y 保存。

Q：账号密码改了怎么办？
A： 删除保存的配置文件（%APPDATA%\xxgc_campus_net_config.txt），然后按第一次的步骤重新设置。

Q：怎么删除保存的配置？
A： 打开文件资源管理器，在地址栏输入 %APPDATA%，找到 xxgc_campus_net_config.txt 文件并删除。
